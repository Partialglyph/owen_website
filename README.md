# owen_website

Personal site — static HTML/CSS, no build step.

## Local preview

Open `index.html` in a browser, or from this folder:

```bash
npx --yes serve .
```

## GitHub Pages

1. Repo → **Settings** → **Pages**
2. **Build and deployment**: Source **Deploy from a branch**
3. Branch **main**, folder **/ (root)** → Save

Site URL: `https://partialglyph.github.io/owen_website/` (after the first deploy finishes).

## Customize

- Copy and edit sections in `index.html`
- Tweak colors and fonts in `styles.css` (`:root` variables at the top)
